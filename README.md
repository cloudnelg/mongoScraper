#Mongo Scraper

A Crudy app that scrapes articles from the New York Times website, user can save and comment on articles.

## Technologies Used

<ul>
    <li>Node.js</li>
    <li>Express.js</li>
    <li> Handlebars.js</li>
    <li> MongoDB</li>
    <li> Mongoose</li>
</ul>

## NPM Packages

<ul>
    <li> body-parser</li>
    <li> express</li>
    <li> express-handlebars</li>
    <li> mongoose</li>
    <li> cheerio</li>
    <li> request</li>
</ul>
